Overview

FireSync is a powerful tool to manage Firebase Realtime Database with a graphical user interface. It simplifies operations like loading, saving, querying, and modifying data.

Key Features

  • Easy Firebase Initialization and Data Loading
  • Intuitive GUI for Data Management
  • Support for CSV Data Import
  • Manage Credentials Securely

Getting Started

Installation Instructions

To get started with FireSync, follow these steps:

  1. Download and install Python from python.org.
  2. Install required libraries using pip: 
    pip install firebase-admin pandas
  3. Download FireSync using this linkMediafireDownloadLink.

Basic Setup

After installation, follow these steps to set up FireSync:

  1. Obtain Firebase credentials from the Firebase Console. See Firebase Setup Guide.
  2. Place the Firebase credentials JSON file in your project directory.
  3. Run FireSync and load the credentials using the provided GUI.

Usage

Loading Firebase Data

After setting up FireSync, you can load data from Firebase Realtime Database. Here’s how:

  1. Open FireSync and select "Load Firebase SDK" from the File menu.
  2. Provide the path to your Firebase credentials JSON file and enter the database URL.
  3. Click "Load Data" to fetch data from Firebase and display it in the tree view.

Adding and Modifying Data

To add or modify data, use the context menu in the tree view:

  • Right-click on a node to open the context menu.
  • Select "Add Data" to add new keys or directories.
  • Select "Edit Data" to modify existing data.
  • Select "Remove Data" to delete data.

Importing Data from CSV

You can import data from a CSV file by selecting "Import CSV" from the context menu.

  1. Select a CSV file with keys and values.
  2. FireSync will add the data to the selected directory.

Advanced Topics

Troubleshooting

If you encounter issues, check the following:

  • Ensure Firebase credentials are correct and valid.
  • Check your internet connection and Firebase database URL.
  • Review error messages for guidance.

Common Issues and Solutions

Initialization Error
Verify the Firebase credentials and database URL.
Data Loading Error
Ensure Firebase SDK is properly initialized.
